# ☁️ Despliegue de Servicios en AWS

Proyecto enfocado en la configuración y despliegue de aplicaciones web en la infraestructura en la nube de Amazon Web Services (AWS). 

## 🛠️ Detalles del Proyecto
El repositorio contiene el código fuente de un servidor Node.js (`index.js`) configurado con sus respectivas dependencias (`package.json`), optimizado para ser alojado en instancias de cómputo en la nube, aplicando principios de escalabilidad y acceso remoto.
